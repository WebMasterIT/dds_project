# Веб-сервис учёта движения денежных средств (ДДС)

Веб-приложение на Django для учёта поступлений и списаний денежных средств.
Позволяет создавать, просматривать, редактировать и удалять записи о движении
средств, вести справочники (статусы, типы, категории, подкатегории) и
фильтровать записи по дате, статусу, типу и категории.

## Возможности

- CRUD записей о ДДС (создание, просмотр, редактирование, удаление)
- Фильтрация списка по периоду дат, статусу, типу, категории и подкатегории
- Управление справочниками через админ-панель
- Логические зависимости: тип → категория → подкатегория
- Динамические выпадающие списки на форме (выбор типа сужает категории,
  выбор категории — подкатегории) через API на Django REST Framework
- Валидация данных на стороне клиента и сервера

## Технологии

- Python 3
- Django 5.2
- Django REST Framework
- SQLite
- Bootstrap 5

## Установка и запуск

1. Клонировать репозиторий:

   git clone https://github.com/ВАШ_ЛОГИН/dds_project.git
   cd dds_project

2. Создать и активировать виртуальное окружение:

   python -m venv venv
   venv\Scripts\activate.bat (Windows)
   source venv/bin/activate (macOS / Linux)

3. Установить зависимости:

   pip install -r requirements.txt

4. Применить миграции (создаст базу данных):

   python manage.py migrate

5. Создать администратора для доступа к админ-панели:

   python manage.py createsuperuser

6. Запустить сервер:

   python manage.py runserver

Приложение будет доступно по адресу http://127.0.0.1:8000/
Админ-панель — по адресу http://127.0.0.1:8000/admin/

## Структура

- `config/` — настройки проекта Django
- `cashflow/` — приложение учёта ДДС (модели, представления, формы, API)
- `cashflow/templates/` — HTML-шаблоны
